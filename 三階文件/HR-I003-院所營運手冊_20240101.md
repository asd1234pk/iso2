## 前言

本手冊旨在為歐亥健復健中心的所有工作人員提供日常營運指南，規範開門前準備、營業中流程以及打烊後收尾工作的標準作業程序。手冊內容涵蓋了院所日常運營的各個環節，旨在確保服務品質一致性、提高營運效率、優化資源使用並確保工作場所安全。

歐亥健復健中心致力於提供高品質的復健服務，優質的服務始於標準化且高效的日常營運。本手冊將幫助每位員工了解並執行自身在日常營運中的職責，共同維護專業、溫馨、高效的醫療服務環境。

## 1. 總則

### 1.1 適用範圍

本手冊適用於歐亥健復健中心各分院的所有工作人員，包括但不限於管理層、治療師、行政人員、客服人員等。

### 1.2 營業時間

- 星期一至星期五：08:00 - 21:00
- 星期六：08:00 - 17:30
- 星期日及國定假日：休診（特殊安排除外）

### 1.3 員工到勤時間

- 早班：07:30 - 16:30
- 中班：11:30 - 20:30  
- 晚班：12:00 - 21:30

### 1.4 責任分工

- **院長/副院長**：監督整體營運，處理突發緊急事件，確保營運品質
- **物理治療師**：負責專業診療服務，維護治療區域整潔與設備狀態
- **客服人員**：負責接待區域管理，預約安排，患者接待與指引
- **行政人員**：負責行政支援，文書處理，資料管理等工作

## 2. 開門前準備

### 2.1 開門流程

#### 2.1.1 首位到達員工職責
1. 解除安全警報系統
2. 開啟大門、接待區與主要區域照明
3. 檢查院所內環境安全（如漏水、電氣異常等）
4. 開啟中央空調系統
5. 開啟服務櫃檯電腦與系統
6. 檢查當日預約情況
7. 填寫「PT-F012-設施設備開關機檢查表」

#### 2.1.2 各區域準備工作

**接待區**
1. 整理等候區座椅、雜誌架與衛教資料
2. 檢查表單存量（如各類同意書、問卷等）
3. 準備飲水機、茶水供應 
4. 清潔接待櫃檯並確認運作正常

**治療區**
1. 檢查並整理各治療室環境
2. 檢查治療設備功能並確認安全可用
3. 補充治療耗材（如電極片、酒精棉片等）
4. 確保毛巾、床單等充足並乾淨
5. 調整適宜室溫與照明

**行政區**
1. 開啟辦公設備（電腦、印表機等）
2. 檢查文具、表單等辦公用品存量
3. 整理公共區域環境

### 2.2 系統啟動

1. 開啟電子病歷系統
2. 確認預約系統正常運作
3. 檢查網路連線狀態
4. 啟動收費系統並確認正常
5. 測試通訊設備（如電話、對講機等）

### 2.3 營業前會議（晨會）

- 時間：營業前15分鐘（07:45-08:00）
- 內容：
  1. 確認當日人員調度與分工
  2. 檢視當日預約情況與特殊個案提醒
  3. 交接前日未完成事項
  4. 設備使用協調
  5. 院長/組長工作指示與要求

## 3. 營業中流程

### 3.1 患者接待流程

#### 3.1.1 初診患者
1. 迎接患者並致歡迎詞
2. 確認預約資訊並引導填寫「PT-F001-患者基本資料表」
3. 說明診所環境與流程
4. 引導患者至等候區
5. 通知治療師患者到達
6. 請患者閱讀並簽署「PT-F006-患者同意書」

#### 3.1.2 複診患者
1. 迎接患者並驗證身份
2. 核對當次預約資訊
3. 更新患者聯絡資料（必要時）
4. 引導患者至等候區
5. 通知治療師患者到達

### 3.2 治療流程管理

1. 治療師按預約時間準時接待患者
2. 治療結束後，及時更新電子病歷系統
3. 治療間隙整理治療室環境與設備
4. 向患者說明下次預約情況
5. 引導患者至前台安排後續預約

### 3.3 預約管理

1. 維護預約系統準確性
2. 預約衝突處理與調整
3. 預約提醒（前一日電話/簡訊提醒）
4. 處理取消或更改預約
5. 記錄並跟進未赴約患者

### 3.4 收費與核銷流程

1. 核對當次療程項目與費用
2. 收取費用並開立收據
3. 協助健保/保險核銷手續
4. 每日中午與晚間進行收費核對
5. 異常處理與調整

### 3.5 營業中設備維護

1. 使用後立即整理設備
2. 定期消毒共用設備表面
3. 發現故障立即報告並停用
4. 記錄設備使用狀況與異常
5. 監控耗材使用並適時補充

### 3.6 環境維護

1. 定時檢查並整理等候區
2. 保持走道暢通與清潔
3. 廁所清潔與耗材補充（至少每2小時檢查一次）
4. 垃圾分類與及時清理
5. 記錄清潔工作於「PT-F011-環境清潔消毒記錄表」

## 4. 打烊後收尾

### 4.1 關店流程

#### 4.1.1 患者離場確認
1. 確認所有患者已離開院所
2. 檢查各治療室與等候區無遺留物品
3. 確認所有約診已完成或妥善安排

#### 4.1.2 系統關閉
1. 備份當日資料
2. 關閉電子病歷系統
3. 關閉預約系統
4. 關閉收費系統
5. 完成當日系統日誌記錄

#### 4.1.3 財務結算
1. 點算現金並與系統核對
2. 整理各類收據與單據
3. 準備存款或交接給負責主管
4. 記錄當日營收情況
5. 完成日結報表

### 4.2 環境整理

1. 所有治療室設備歸位與清潔
2. 公共區域清潔與整理
3. 垃圾集中處理並丟棄
4. 準備隔日所需耗材與表單
5. 執行感染控制相關消毒程序
6. 填寫「PT-F011-環境清潔消毒記錄表」

### 4.3 設備關閉

1. 關閉所有治療設備電源
2. 檢查所有電器設備是否關閉
3. 關閉空調系統
4. 關閉不必要照明
5. 填寫「PT-F012-設施設備開關機檢查表」

### 4.4 安全檢查

1. 檢查所有窗戶是否關閉與鎖好
2. 檢查水龍頭是否關閉
3. 檢查重要區域門窗是否上鎖
4. 檢查貴重物品是否妥善保管
5. 設定安全警報系統
6. 最後離開者鎖門並確認門窗安全

### 4.5 交接事項

1. 填寫工作交接日誌
2. 記錄當日特殊事件或未完成事項
3. 準備隔日需注意事項
4. 必要時與接班人員電話交接
